## 📊 Blinkit Sales Analysis Project
## 🧾 Project Overview
This project focuses on conducting a comprehensive analysis of Blinkit’s sales performance, 
customer satisfaction, and inventory distribution.
The analysis is performed using Python in a Jupyter Notebook environment, with the 
objective of identifying key insights, trends, and optimization opportunities through 
KPIs and visualizations.
___________________________________________________________________________________________
## 🎯 Business Objective
To analyze Blinkit’s operational data and answer key business questions related to:
•	Sales performance
•	Customer preferences
•	Outlet-level performance
•	Product-level insights
This analysis helps stakeholders make data-driven decisions to improve sales, inventory 
management, and customer satisfaction.
___________________________________________________________________________________________
## 🧮 KPIs Analyzed
The following Key Performance Indicators (KPIs) were calculated and analyzed:
•	Total Sales – Overall revenue generated from all items sold
•	Average Sales – Average revenue per sale
•	Number of Items – Total count of different items sold
•	Average Rating – Average customer rating of sold items
